## 环境配置和软件安装

使用install命令可以轻松安装常用软件和一键配置环境的功能

### 查看当前支持的环境配置和软件

**输入命令**

```bash
xlings install
```

**显示结果**

```bash
	xinstaller lists
 mdbook
 g++
 python3
 vs
 python
 vscode
 cpp
 c
 gcc
 c++
```

### 一键环境配置

配置C语言环境

```bash
xlings install c
```

配置Python环境

```bash
xlings install python
```

### 一键安装软件

安装vscode编辑器

```bash
xlings install vscode
```

安装VS(visual studio)

```bash
xlings install vs
```
